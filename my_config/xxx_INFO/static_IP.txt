 
== 'výpis nastaveni' ==
 nmcli 
 nmcli device show     ' to get complete information about known devices '
 nmcli connection show   ' to get an overview on active connection profiles '


== 'výpis wifi sítí'  
  nmcli dev wifi list
  

== pro "wlan0" == 

sudo nmcli con mod "MFG" ipv4.addresses "192.168.0.34/24"
sudo nmcli con mod "MFG" ipv4.gateway "192.168.0.1"
sudo nmcli con mod "MFG" ipv4.dns "192.168.0.1"
sudo nmcli con mod "MFG" ipv4.method manual
sudo nmcli con down "MFG" && sudo nmcli con up "MFG"
  


----------------------------------------


== 'pokud to nepůjde' 

sudo nano /etc/dhcpcd.conf

interface wlan0
static ip_address=192.168.0.34/24
static routers=192.168.0.1
static domain_name_servers=192.168.0.1


sudo systemctl restart dhcpcd

přidání další Wifi

1. Přímo v system.cfg (pokud to skript podporuje)

## wifi name
WIFI_SSID="MFG"
## wifi password
WIFI_PASSWD="72212209"

## second wifi
WIFI_SSID2="Domaci_WiFi"
WIFI_PASSWD2="heslo123"

Ale: Musíš zkontrolovat, jestli instalační skript CB1 tohle podporuje. Není to standardní.


2. Pomocí NetworkManager (doporučuji)
Spolehlivější je přidat druhou WiFi ručně přes nmcli:

# Přidej druhou WiFi síť

sudo nmcli dev wifi connect "Nazev_Druhe_Wifi" password "heslo_druhe_wifi"


# Nastav priority (vyšší číslo = vyšší priorita)

sudo nmcli con mod "MFG" connection.autoconnect-priority 10
sudo nmcli con mod "Nazev_Druhe_Wifi" connection.autoconnect-priority 5
